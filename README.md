## System Overview

Designed a new information system with the following core modules:

- **Point Wallet** — customers earn points from purchases, services, or donations
- **Partner Management** — shops and animal hospitals select point packages and set conditions
- **Point Redemption** — redeem points for discounts or donate to stray animal foundations
- **Adoption System** — browse and request adoption of stray animals through the platform
- **Notification System** — real-time alerts for point transactions and adoption status
